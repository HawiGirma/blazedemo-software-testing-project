## Title: Missing Input Validation

## Description:
The system allows users to submit the booking form without entering required fields.

## Steps to Reproduce:
1. Open booking page
2. Leave name empty
3. Click "Purchase Flight"

## Expected Result:
System should show an error message

## Actual Result:
Booking is processed successfully

Medium
