# BlazeDemo Software Testing Project

## 📌 Project Title
Functional and Usability Testing of BlazeDemo Flight Reservation System

## 📖 Overview
This project focuses on testing the BlazeDemo web application using manual testing techniques. The objective is to evaluate system functionality, usability, and identify defects.

## 🧪 Testing Scope
- Flight search functionality
- Flight selection
- Booking process
- Form validation

## 🔍 Testing Approach
- Manual Testing
- Black-box Testing
- Functional Testing

## 📂 Project Structure
- docs → Final report
- test-cases → Designed test cases
- test-results → Execution results
- bug-reports → Identified bugs
- screenshots → Evidence of testing

## 🐞 Key Bug Found
The system allows booking submission without required input fields (e.g., name), indicating missing validation.

## 🛠 Tools Used
- Browser (Chrome)
- Manual testing techniques

## 📊 Result Summary
Most core functionalities passed, but validation and error handling require improvement.
